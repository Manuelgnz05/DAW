Estructura Estándar de un Documento HTML

Esquema visual que muestra la jerarquía y relación entre las etiquetas semánticas de HTML5

Código HTML Estructurado

<!DOCTYPE html>
<html lang="es">
  <head>
    <!-- Metadatos, enlaces a CSS, título -->
  </head>
  
  <body>
    <header>
      <!-- Encabezado: logo, título principal -->
      <nav>
        <!-- Menú de navegación -->
      </nav>
    </header>
    
    <main>
      <!-- Contenido principal -->
      <section>
        <!-- Sección temática -->
        <article>
          <!-- Contenido independiente -->
        </article>
        <article>
          <!-- Otro contenido independiente -->
        </article>
      </section>
      
      <aside>
        <!-- Contenido relacionado (barra lateral) -->
      </aside>
    </main>
    
    <footer>
      <!-- Pie de página -->
    </footer>
  </body>
</html>

Explicación de las Etiquetas Semánticas

<header>

Representa el encabezado de una página o sección. Suele contener el logotipo, título principal y menú de navegación.

Contiene: Logo, título principal, menú de navegación (<nav>)

<nav>

Define un bloque de enlaces de navegación. Debe usarse para el menú principal del sitio.

Contiene: Listas de enlaces (<ul>, <li>, <a>)

<main>

Representa el contenido principal del documento. Solo debe haber un <main> por página.

Contiene: <section>, <article>, <aside>

<section>

Agrupa contenido temáticamente relacionado. Cada sección debe tener su propio encabezado.

Contiene: <article>, <header> (opcional), contenido relacionado

<article>

Representa contenido independiente y autónomo que podría distribuirse por separado.

Contiene: Texto, imágenes, videos, etc. que forman una unidad de contenido

<aside>

Contiene contenido relacionado indirectamente con el contenido principal, como barras laterales.

Contiene: Información complementaria, enlaces relacionados, publicidad

<footer>

Representa el pie de página de una página o sección. Suele contener información de contacto, enlaces legales, etc.

Contiene: Información de contacto, enlaces legales, mapa del sitio, redes sociales
